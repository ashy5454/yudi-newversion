# Yudi Landing Page - PRD

## Project Overview
Yudi is an emotional AI companion app that speaks multiple Indian languages. The landing page showcases its Gen-Z focused features with creative, animated UI.

## Original Problem Statement
Redesign the Yudi landing page with:
1. Hero section split into text + motion graphic video showing how to use Yudi
2. New features: "Talk like you actually text", "Gen Z brain not textbook AI", "Clingy in best way", "Build a soul from scratch"
3. Horizontal auto-swipe carousel for features with Gen Z images
4. 3D animated language blocks with heavy animation
5. Faster flip words animation (2 seconds)
6. New FAQs with creative contact styling (hearts/animation)
7. Slimmer footer
8. Improved header and title bar styling
9. Improved Create Persona page styling

## User Personas
- **Primary**: Indian Gen-Z (18-25) college students
- **Secondary**: Young professionals who want emotional support in their native language

## Core Requirements (Static)
- Dark starry theme with purple/pink accent colors
- Multi-language support (English, Hindi, Telugu, Tamil, etc.)
- Gen-Z language and slang throughout
- Responsive design
- Smooth animations using Framer Motion
